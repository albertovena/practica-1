W3cschools



type color

Lo que hace es utilizar campos de entrada que tienen que contener un color, también nos muestra los diferentes colores existentes

type date

Se utilizan para campos que deben de contener una fecha, puede aparecer un selector de fecha en el campo de entrada

type datetime-local

Se utiliza igual que en el de type date pero te permite seleccionar también una franja horaria

type email

Te obliga para que el texto introducido tenga si o si la estructura que tiene un correo electrónico, algunos telefonos reconocen el correo y agregan el .com

type month

Permite al usuario seleccionar solo un mes y un año, además de poder seleccionar directamente el mes en el que estamos

type number

Podemos seleccionar un numero en un intervali de cantidades que nosotros pongamos

type range

Se usa para ingresar un numero cuyo valor exacto no es importante

type search

Se utiliza para campos de búsqueda

type tel

Se utiliza para campos de entrada que deben contener un número de teléfono

type time

Permite al usuario seleccionar una hora exacta(sin zona horaria)

type url

Se utiliza para campos que deben de contener direcciones URL

type week

Permite al usuario seleccionar una semana y un año
